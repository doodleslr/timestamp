Timestamp convert to Express

User Stories:

  1. A date string is valid if can be successfully parsed by new Date(date_string).
  2. If the date string is empty it will provide the current date and time.
  3. If the date string is valid it will be parsed through the new Date() method and return in an appropriate format.
  4. If the date string is invalid it will say so.

For Example:

The date requested is: Thu Mar 07 2019 11:35:40 GMT+1100 (Australian Eastern Daylight Time)
